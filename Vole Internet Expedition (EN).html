Vole Internet Expedition

Vole Internet Expedition is a new generation web browser that brings a faster and more elegant web browsing experience. All websites can be displayed in a beautiful gallery format with one click. It is a super-fast web browser with multiple screens and layouts that can be saved and restored.

Launched in 2013. It has been repeatedly selected by the famous American software sales website BITSDUJOUR Recommended, with a fairly high popularity . Microsoft Edge has copied it many times, but still cannot surpass it.

Basic Functions

Browser kernel

Same as Microsoft Edge and Google Chrome Chromium browser kernel.

 

Browser Features

Supports web browsing, video playback, search, downloading, password saving and auto-fill, as well as Microsoft Edge extensions. History management, favorites management. Supports 4K displays.

 

Unique Features

Topology

Through the 5-layer topology, it manages massive URLs for you, allowing you to easily browse bulk web pages without typing a single address character.

 

Saving and restoring layouts

Supports tiled, overlapping and combined layouts, making dual-screen and multi-screen browsing easy. Layouts can be manually exported and imported, as well as automatically loaded at startup. You no longer have to frequently enter URLs and adjust web page layouts.

 

Browser extensions

Automatically identify installed EDGE browser extensions. Users can customize which EDGE browser extensions to enable.

 

Multi-engine parallel search

Multiple search engines search simultaneously, and multiple split screens display search results for easy comparison. Preset commonly used search engines and custom search engines.

 

Scene Export

Save the complete browser layout and URL data to the exported file. Can be imported by this machine and other computers.

 

Scene Import

Import the scene file to restore the complete layout and URL.

 

Global Navigation

More than 180 countries and regions, more than 40,000 websites for you to browse easily.
