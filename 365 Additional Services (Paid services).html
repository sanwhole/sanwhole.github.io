365 Additional Services (Paid services)

 

Vole Office 365 server is permanently free software. Therefore, you can use the suite without paying. Your payment will not increase the software functions, but you will get additional services as follows:

Represent is now possible. The software will display the user's logo and username, making it more professional and polished.

Usability is guaranteed. Technical support can be obtained to make commercial use more reliable.

Upgrade is guaranteed. Paid software upgrades are guaranteed.

User compliance is guaranteed. Because user access, usage, and upgrades are all guaranteed, user compliance is automatically demonstrated. Please note that Vole Office free software users cannot obtain compliance protection.

After payment, please modify the 365.cfg file in the App_Data directory to set a custom name. The modified settings will take effect after the website restarts.
