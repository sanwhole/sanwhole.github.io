Vole Windows Expedition

Multi-screen, built-in previewer, file explorer for editors

Launched in 2013. It has been repeatedly selected by the famous American software sales website BITSDUJOUR Recommended, with a fairly high reputation . Microsoft Windows 11 has copied it many times , but still cannot surpass it.

Basic Functions

Basic File Explorer

It uses the original Microsoft File Explorer control, so it is exactly the same as the operating system's File Explorer.

It has all the functions of Microsoft Windows File Explorer. Please refer to Microsoft system help for specific functions, which will not be described here.

Resource management group layout automatic saving and automatic restoration

The software provides 1 single file explorer, 4 dual-screen, 1 triple-screen, and 1 quad-screen file explorer group, for a total of 16 independent file explorers. The layout of each file explorer and the files being previewed will be automatically saved when the software is exited, and automatically restored when it is run next time.

 

Built-in file previewer

Each independent file explorer has a built-in file previewer. When you select a file, the previewer will preview the file in the built-in previewer at the same time without opening other software. The built-in file previewer supports multiple file formats, including: pictures, audio, video, text files, RTF, WORD, EXCEL, PDF, GIF, HTML, and Diagram, etc.

File Comments

You can add comments to any file without changing the original file. Comments are automatically displayed when you preview the file. Comments are hosted by the operating system and are invisible to other software. Moreover, since the comment process does not modify the file in any way, even Microsoft Explorer cannot view the comments. Editing and renaming the file will not affect the file comments.

In actual testing, the operating system's hosting of comment content is not perfect. If the file is moved, copied, or pasted on the local computer, the comment content will not be lost. However, if the file is copied or moved to another computer, the file comment content may be lost. Whether the file comment is lost during the file operation depends on the file operation method. What is certain is that if the carrier of the file operation is the network, the file comment content will definitely be lost. If the carrier of the file operation is a USB flash drive or a mobile hard drive, the file comment content will not be lost. [MP4]

Directory Tracking

The directory tracking function gives you a bird's eye view of the current directory of all file resource management. And it will be marked with different colors according to the time when the directory was opened. You can expand or collapse the directory tracking window by clicking the directory tracking button.
