Vole Windows Expedition【IMG】是新一代文件管理器，是一种快速、更优雅的文档管理方式。Vole Windows Expedition拥有多个文件资源管理器组【IMG】，每个文件资源管理器组包含多个文件资源管理器【IMG】。每个文件资源管理器都可以独立的浏览文件目录【IMG】和预览文件内容【IMG】或是编辑文件【IMG】。不论是预览文件还是编辑文件都是在内嵌的预览器或编辑器中进行，而不必打开其它软件。

Vole Windows Expedition目录跟踪功能，可以鸟瞰当前所有文件资源管理的当前目录。而且会依据目录打开时间用不同的颜色标识。通过点击目录跟踪按钮可以展开或收起目录跟踪窗口【IMG】。

Vole Windows Expedition 包括1个单资源管理器组【IMG】，4个双资源管理器组【IMG】，1个3资源管理器组【IMG】和1个4资源管理器组【IMG】。其中，个单资源管理器组，双资源管理器组，3资源管理器组和4资源管理器组，分别含有1个，2个，3个和4个独立的文件资源管理器。因此，Vole Windows Expedition共计包含16个独立的文件资源管理器。

资源管理器组中的多个文件资源管理器是完全独立的。通过调整分隔栏可以调节文件资源管理器的占比。通过最大化按钮可以将任意一个文件资源管理器最大化【IMG】或是由最大化恢复到原始状态。

在预览文件时资源管理器组中的多个文件资源管理器还可以协同预览。在一个资源管理中选中的一个文件可以在组内的全部资源管理器中显示。在预览图片时还可以自动在组内其它文件资源管理器显示镜像图片【IMG】。在预览视频时也可以自动在组内其它文件资源管理器显示视频【IMG】。镜像视频和镜像图片可以为你带来不一样的用户体验。

每个文件资源管理器包含一个文件浏览器【IMG】， 1个文件预览器【IMG】和1个文件编辑器【IMG】。其中，文件浏览器与Windows 文件资源管理器100%相同。文件预览器用于预览在文件浏览器中选中的文件。文件编辑器用于编辑在文件浏览器中选中的文件。文件预览器与文件编辑器共用同一窗口，在预览状态时显示预览器内容，在编辑状态时加载编辑控件。

文件资源管理器和文件预览器都有记忆功能。在软件关闭时会自动记录打开的目录和预览的文件。当再次打开软件时可以自动打开关闭时的目录和预览的文件。如果不希望回复打开的文件，可以在关闭软件前点击预览器中的小刷刷图标清除预览文件记录即可。

内置的文件预览器可以在文件浏览器选中文件时自动预览该文件的内容，而不必打开其它软件。内置的文件预览器支持多种文件格式，包括：图片【IMG】，音频，视频【IMG】，文本文件，RTF，WORD【IMG】, EXCEL, PDF, GIF, HTML，以及Diagram等。

内置的文件编辑器可以在文件浏览器选中文件时自动编辑该文件，而不必打开其它软件。内置的文件编辑器支持多种文件格式，包括：图片，文本文件，RTF，Word, Excel, 以及Diagram等。

默认文件预览器显示在文件浏览器上面。用户可以点击文件预览器位置设置按钮调整文件预览器显示位置。四个文件预览器位置设置按钮分别将文件预览器显示在文件浏览器的上方【IMG】，下方【IMG】，左侧【IMG】和右侧【IMG】。

默认文件预览器显示和文件浏览器显示各占50%。选择文件预览器显示占比按钮可以调整文件预览器显示占比从10%到90%【IMG】。

点击文件预览器最大化按钮，可以将文件预览器最大化，此时文件浏览器将自动隐藏【IMG】。

点击文件浏览器最大化按钮，可以将文件浏览器最大化，此时文件预览器将自动隐藏【IMG】。

点击文件资源管理器最大化按钮，可以将文件资源管理器最大化，此时文件资源管理器组中的其它文件资源管理器将自动隐藏【IMG】。

文件预览器默认会预览用户选中 的文件。如果不想预览文件，可以选择暂停预览按钮。暂停预览按钮被选中的情况下预览器将暂停预览选中的文件。再次点击该按钮将又恢复自动预览状态【IMG】。

预览器预览文件时可能会独占文件，造成其它软件无法使用该文件。此时，点击清理预览器按钮可以让预览器彻底释放该文件【IMG】。

点击目录复制按钮可以复制当前目录，然后在其它文件浏览器点击目录黏贴按钮就可以快速打开该目录【IMG】。

文件预览器可以自动播放当前目录下的所有音视频文件。播放按钮用于播放或暂停播放。循环按钮用于设置循环播放。从新加载播放按钮用于更新了目录中的媒体文件后更新播放列表【IMG】。

镜像图片和视频制作。选中一个图片或视频后可以通过媒体工具中的镜像工具制作水平镜像，垂直镜像和水平垂直镜像图片或视频【IMG】。

视频转MP4。该功能可以将任何视频转为MP4【IMG】。

改变MP4尺寸。该功能可以将视频修改为用户指定的尺寸【IMG】。

当前目录可以添加到目录收藏夹。软件预设7个收藏夹目录：综合，工作，学习，休闲，生活，工具，VMC，方便用户管理收藏夹。目录收藏夹在所有文件浏览器内共享【IMG】。

文本文件的识别不是根据文件的扩展名来确定的，而是通过软件自动分析文件内容，智能判别文本文件类型。因此，真实的以TXT为扩展名的文件，以及真实的C、CPP、CS、CSS、JS等文件都会被自动识别为文本文件。而即使将MP4文件的扩展名改为TXT也不会被识别为文本文件。

文本文件中网址的自动识别与网页浏览。预览文本文件时，文本预览器自动分析文件内容，找出文件中所有网址并以列表中形式供用户打开相应的网页。用户可以通过在网址列表中选择网址就可以在内置的网页浏览器中打开相应的网页。【IMG】

 

图片预览器支持全景显示【IMG】，缩放显示【IMG】和伸展显示【IMG】。

全景显示模式下，图片将保持原图的横纵比例，将图片最大可能的在预览窗中显示。

缩放显示模式下，点击图片后，可以通过鼠标中键向上滚动放大，向下滚动缩小显示图片。图片的缩放比例也可以通过右键菜单【IMG】直接选择。预览器右下角图片预览控制窗口将显示当前预览位置的缩略图。通过拖动缩略图位置，或者点中图片后直接拖动图片都可以查看图片不同位置。

伸展显示模式下，图片将不再保持原图的横纵比例，而将图片伸展填满整个预览窗口。

图片预览除了支持单预览功能外还支持多浏览器协同预览。比如，自动在浏览组中的其它浏览器预览该图片的镜像图片。请参阅相关章节。

音视频预览器核心播放控件使用内嵌微软Windows Media Player作为媒体播放器。具备所有微软Windows Media Player既有功能。比如视频缩放，全屏播放等。另外，音视频预览器还支持A-B段复读功能，循环播放功能。

视频预览除了支持单预览功能外还支持多浏览器协同预览。比如，自动在浏览组中的其它浏览器预览该视频的镜像视频。请参阅相关章节。

你可以通过安装免费的解码器文件（比如：K-Lite Codec Pack）增强内嵌的微软Windows Media Player播放各种音视频 文件的能力。比如：TS视频， VOB视频等，这些视频在微软Windows 10默认是不被支持的。

默认预览器预览文件时是相互独立进行的。协同预览功能可以协调文件资源管理器组中多个文件资源管理器的预览器协同工作，让用户在一个预览器中选择文件，同时在多个预览器中预览文件。协同预览有手动协同预览和自动协同预览 。

勾选协同预览按钮既可以进入手动协同预览模式。如果用户选择一个文件，则在组内的其它预览器都将预览该文件【IMG】。请注意:3预览器组的中间预览器从不参加协同预览【IMG】。如果用户选择多余一个文件，则组内的其它预览会先全部预览第一个选中文件。然后，第二个预览器预览第二选中文件， 第三预览器预览第三选中的文件（如果有第三选中文件）【IMG】，以此类推。

勾选协同预览按钮同时再勾选镜像按钮，如果再选择图片将进入图片自动镜像预览模式。图片的自动镜像预览模式仅对图片有效，且仅需选中一个图片。在图片自动镜像预览模式下，预览图片时会自动在组内的其它预览器中预览该图片的镜像图片。该功能对2资源管理器组，3资源管理器组和4资源管理器组有效。

勾选协同预览按钮同时再勾选镜像按钮，如果再选择视频将进入视频自动镜像预览模式。视频自动镜像预览模式仅对视频有效，且仅需选中一个视频。在视频自动镜像预览模式下，预览视频时会自动在组内的其它预览器中预览该视频的镜像视频。该功能对2资源管理器组，3资源管理器组和4资源管理器组有效。请注意：视频自动镜像预览模式要求在主视频相同目录下必须预先准备好镜像视频。而且镜像视频命名必须满足如下要求。

镜像视频的命名规则: 假定原视频文件名为: 原视频.mp4,  则水平镜像为：原文件名_HF.mp4,  垂直镜像为：原文件名_VF.mp4,  垂直水平镜像为：原文件名_VH.mp4。

视频自动镜像预览模式下, 2资源管理器组，3资源管理器组需要准备水平镜像文件。4资源管理器组需要准备全部3个镜像文件。

镜像视频的制作可以通过文件浏览器中的工具按钮菜单制作。
