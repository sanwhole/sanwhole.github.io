Vole Windows Expedition是新一代文件管理器，是一种快速、更优雅的文档管理方式。Vole Windows Expedition有双资源管理器、三资源管理器和四资源管理器，总共有16个资源管理器与微软文件资源管理器完全兼容。每个浏览器的内置文档预览器都可以让您在内置媒体播放器中查看各种资源，包括图像、音频、视频、Word、Excel、PDF、RTF、TXT、GIF、HMTL等。协作预览功能允许您轻松同时预览图像的水平和垂直翻转。同样，在预览视频时，请同时以所有四个翻转角度预览原始视频。这将为您带来全新的图片和视频体验。

Vole Windows Expedition 包括1个单资源管理器组，4个双资源管理器组，1个3资源管理器组和1个4资源管理器组。其中，个单资源管理器组，双资源管理器组，3资源管理器组和4资源管理器组，分别含有1个，2个，3个和4个独立的文件资源管理器。因此，Vole Windows Expedition共计包含16个独立的文件资源管理器。

资源管理器组中的多个文件资源管理器是完全独立的。通过调整分隔栏可以调节文件资源管理器的占比。通过最大化按钮可以将任意一个文件资源管理器最大化或是由最大化恢复到原始状态。

在预览文件时资源管理器组中的多个文件资源管理器还可以协同预览。在一个资源管理中选中的一个文件可以在组内的全部资源管理器中显示。在预览图片时还可以自动在组内其它文件资源管理器显示镜像图片。在预览视频时也可以自动在组内其它文件资源管理器显示视频。镜像视频和镜像图片可以为你带来不一样的用户体验。

每个文件资源管理器包含一个文件浏览器， 1个文件预览器和1个文件编辑器。其中，文件浏览器与Windows 文件资源管理器100%相同。文件预览器用于预览在文件浏览器中选中的文件。文件编辑器用于编辑在文件浏览器中选中的文件。文件预览器与文件编辑器共用同一窗口，在预览状态时显示预览器内容，在编辑状态时加载编辑控件。

文件资源管理器和文件预览器都有记忆功能。在软件关闭时会自动记录打开的目录和预览的文件。当再次打开软件时可以自动打开关闭时的目录和预览的文件。如果不希望回复打开的文件，可以在关闭软件前点击预览器中的小刷刷图标清除预览文件记录即可。

内置的文件预览器可以在文件浏览器选中文件时自动预览该文件的内容，而不必打开其它软件。内置的文件预览器支持多种文件格式，包括：图片，音频，视频，文本文件，RTF，Word, Excel, PDF, GIF, HTML，以及Diagram等。

内置的文件编辑器可以在文件浏览器选中文件时自动编辑该文件，而不必打开其它软件。内置的文件编辑器支持多种文件格式，包括：图片，文本文件，RTF，Word, Excel, 以及Diagram等。

软件自动分析文件，智能判别文本文件类型，而不是仅依据文件扩展名。预览文本文件时，软件自动分析文件内容，找出文件中所有网址并在网址列表中显示。用户可以通过在网址列表中选择网址在内置的网页浏览器中打开相应的网页。
