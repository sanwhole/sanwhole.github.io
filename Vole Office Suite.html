System requirements, download, use and upgrade

System requirements

Support for Windows 7 and higher operating systems. Windows 7, Windows 8.1 require .NET Framework 4.7.2 or higher to be installed.

Vole Internet Expedition and Vole 4M Showroom require Windows 10 or higher.

Download

Download address: here or here.

The download file name is VoleOffice.exe. It is a standard portable single EXE file, about 118 MB in size, and does not require installation. It can also be used in a USB flash drive.

Use

run VoleOffice.exe directly.

It is recommended to put VoleOffice.exe in the Vole Office directory for use. In this way, the user data generated when the software is running will also be created in this directory.

Upgrade

When the software is running, it will automatically detect the latest version and prompt the user to upgrade. To upgrade the software, you only need to replace the downloaded VoleOffice.exe. Since this software is permanently free software, it can be upgraded permanently for free.

It is recommended that you update to the latest version at any time. The software sets a minimum usable software version number, and the software will not work properly if it is lower than this version. As the software is upgraded, the minimum usable software version number will continue to increase. Therefore, you cannot use the current version permanently.
