Vole Windows Expedition【IMG】is a new generation of file manager, a faster and more elegant way to manage documents. Vole Windows Expedition has multiple file explorer groups【IMG】, each of which contains multiple file explorers【IMG】. Each file explorer can independently browse file directories【IMG】, preview file contents【IMG】, or edit files【IMG】. Whether previewing or editing files, it is done in the embedded previewer or editor without opening other software.

Vole Windows Expedition includes 1 single explorer group 【IMG】, 4 dual explorer groups 【IMG】, 1 3 explorer group 【IMG】 and 1 4 explorer group 【IMG】. Among them, the single explorer group, dual explorer group, 3 explorer group and 4 explorer group contain 1, 2, 3 and 4 independent file explorers respectively. Therefore, Vole Windows Expedition contains a total of 16 independent file explorers.

Multiple file explorers in the explorer group are completely independent. The proportion of file explorers can be adjusted by adjusting the separator bar. Any file explorer can be maximized 【IMG】 or restored to its original state by the maximize button.

Multiple file explorers in the explorer group can also preview together when previewing files. A file selected in one explorer can be displayed in all explorers in the group. When previewing pictures, mirrored pictures can also be automatically displayed in other file explorers in the group [IMG]. When previewing videos, videos can also be automatically displayed in other file explorers in the group [IMG]. Mirror videos and mirror pictures can bring you a different user experience.

Each file explorer contains a file browser [IMG], a file previewer [IMG] and a file editor [IMG]. Among them, the file browser is 100% the same as the Windows file explorer. The file previewer is used to preview the file selected in the file browser. The file editor is used to edit the file selected in the file browser. The file previewer and the file editor share the same window, displaying the previewer content in the preview state and loading the editing control in the editing state.

Both the file explorer and the file previewer have a memory function. When the software is closed, the opened directory and previewed files will be automatically recorded. When the software is opened again, the directory and previewed files at the time of closing can be automatically opened. If you do not want to restore the open file, you can click the small brush icon in the previewer before closing the software to clear the preview file record.

The built-in file previewer can automatically preview the contents of the file when the file is selected in the file browser without having to open other software. The built-in file previewer supports a variety of file formats, including: pictures 【IMG】, audio, video 【IMG】, text files, RTF, WORD [IMG], EXCEL, PDF, GIF, HTML, and Diagram.

The built-in file editor can automatically edit the file when the file is selected in the file browser without opening other software. The built-in file editor supports a variety of file formats, including: pictures, text files, RTF, Word, Excel, and Diagram.

The default file previewer is displayed above the file browser. Users can click the file previewer position setting button to adjust the file previewer display position. The four file previewer position setting buttons display the file previewer above the file browser 【IMG】, below 【IMG】, on the left 【IMG】, and on the right 【IMG】.

The default file previewer display and file browser display each account for 50%. Select the file previewer display ratio button to adjust the file previewer display ratio from 10% to 90% 【IMG】.

Click the file previewer maximize button to maximize the file previewer, and the file browser will automatically hide 【IMG】.

Click the file browser maximize button to maximize the file browser, and the file previewer will automatically hide 【IMG】.

Click the Maximize File Explorer button to maximize the file explorer, and other file explorers in the file explorer group will be automatically hidden [IMG].

The file previewer will preview the file selected by the user by default. If you do not want to preview the file, you can select the Pause Preview button. When the Pause Preview button is selected, the previewer will pause previewing the selected file. Clicking the button again will restore the automatic preview state [IMG].

When previewing a file, the previewer may occupy the file exclusively, making it impossible for other software to use the file. At this time, click the Clean Previewer button to allow the previewer to completely release the file [IMG].

Click the Copy Directory button to copy the current directory, and then click the Paste Directory button in other file browsers to quickly open the directory 【IMG】.

The file previewer can automatically play all audio and video files in the current directory. The Play button is used to play or pause playback. The Loop button is used to set loop playback. The Reload Play button is used to update the playlist after updating the media files in the directory [IMG].

The current directory can be added to the directory favorites. The software presets 7 favorite directories: General, Work, Study, Leisure, Life, Tools, VMC, which is convenient for users to manage favorites. Directory favorites are shared in all file browsers [IMG].

The identification of text files is not determined by the file extension, but by the software automatically analyzing the file content and intelligently identifying the text file type. Therefore, real files with TXT extensions, as well as real C, CPP, CS, CSS, JS and other files will be automatically identified as text files. Even if the extension of an MP4 file is changed to TXT, it will not be identified as a text file.

Automatic identification and web browsing of URLs in text files. When previewing a text file, the text previewer automatically analyzes the file content, finds all URLs in the file and lists them for users to open the corresponding web pages. Users can open the corresponding web page in the built-in web browser by selecting a URL in the URL list. [IMG]

 

The image previewer supports panoramic display [IMG] , zoom display [IMG] and stretch display [IMG] .

In panoramic display mode, the image will maintain the horizontal and vertical ratios of the original image and display the image in the preview window as large as possible.

In zoom display mode, after clicking on the image, you can use the middle mouse button to scroll up to zoom in, and scroll down to zoom out. The zoom ratio of the image can also be directly selected through the right-click menu [IMG] . The image preview control window in the lower right corner of the previewer will display a thumbnail of the current preview position. You can view different positions of the image by dragging the thumbnail position, or by directly dragging the image after clicking on the image.

In the stretch display mode, the image will no longer maintain the original horizontal and vertical ratios, but will stretch to fill the entire preview window.

In addition to single preview, image preview also supports multi-browser collaborative preview. For example, the image will be automatically previewed as a mirror image in other browsers in the browsing group. Please refer to the relevant section.

By default, previewers preview files independently. The collaborative preview feature can coordinate the previewers of multiple file explorers in a file explorer group to work together, allowing users to select files in one previewer and preview them in multiple previewers at the same time. Collaborative preview includes manual collaborative preview and automatic collaborative preview.

Different from manual collaborative preview, automatic collaborative preview automatically prepares the collaborative preview environment, such as preparing the previewer window, hiding the corresponding toolbar, loading the corresponding media, etc.

Select a picture and click the Prepare command to automatically preview the picture mode. This function is valid for 2-Explorer Group [IMG], 3-Explorer Group [IMG] and 4-Explorer Group [IMG].

Select the video and click the Prepare command to automatically preview the video mode. This function is valid for 2-resource manager group [IMG]【MP4】, 3-resource manager group [IMG] [ MP4] and 4-resource manager group [IMG]【MP4】. Please note: The automatic video mirror preview mode requires that the mirror video must be prepared in advance in the same directory as the main video.

Automatic collaborative preview commands include: prepare, play, pause, stop, synchronize, jump here to play, cancel, etc. Among them:

Prepare command: After selecting an image or video, first select the prepare command to let the software automatically prepare the necessary preview environment, such as maximizing the previewer, hiding the toolbar, and automatically loading the corresponding media and mirror media in the previewer. [IMG] [ MP4]

Cancel command: Restore the original settings of the previewer and cancel media preview.

Jump here to play command: Synchronize the playback progress of all previewers. This function can not only play the video at the specified position, but also synchronize the progress of multiple players.

Check the Collaborative Preview button to enter manual collaborative preview mode. If the user selects a file, all other previewers in the group will preview that file [IMG]. This feature is valid for 2-explorer group [IMG], 3-explorer group [IMG] and 4-explorer group [IMG]. Please note: the middle previewer in a 3-previewer group never participates in collaborative preview [IMG]. If the user selects more than one file, all other previewers in the group will preview the first selected file first. Then, the second previewer previews the second selected file [IMG], the third previewer previews the third selected file (if there is a third selected file) [IMG], and so on.

Check the collaborative preview button and then check the mirror button [IMG]. If you select a picture, you will enter the automatic mirror preview mode. The automatic mirror preview mode of the picture is only valid for pictures, and only one picture needs to be selected. In the automatic mirror preview mode of the picture, when you preview a picture, the mirror image of the picture will be automatically previewed in other previewers in the group. This function is valid for 2-explorer group [IMG], 3-explorer group [IMG] and 4-explorer group [IMG].

Check the collaborative preview button and then check the mirror button. If you select a video, you will enter the video automatic mirror preview mode. The video automatic mirror preview mode is only valid for videos, and only one video needs to be selected. In the video automatic mirror preview mode, when previewing a video, the mirror video of the video will be automatically previewed in other previewers in the group. This function is valid for 2 resource manager groups [IMG] [ MP4], 3 resource manager groups [IMG] [ MP4] and 4 resource manager groups [IMG] [ MP4]. Please note: The video automatic mirror preview mode requires that the mirror video must be prepared in advance in the same directory as the main video. And the mirror video naming must meet the following requirements.

Naming rules for mirrored videos: Assuming the original video file name is: original video.mp4, the horizontal mirror image is: original file name_HF.mp4, the vertical mirror image is: original file name_VF.mp4, and the vertical and horizontal mirror images are: original file name_VH.mp4. [IMG]

In the video automatic mirror preview mode, the 2nd and 3rd resource manager groups need to prepare horizontal mirror files. The 4th resource manager group needs to prepare all 3 mirror files.

Mirror video can be created through the tool button menu in the file browser.

If the carrier of the file operation is a USB flash drive or a movable hard disk, the file review content will not be lost.
